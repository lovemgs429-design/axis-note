# ◢ aXIs NOTE 🦋⛓
### UNOFFICIAL fan-made DATABASE for "Utawaku"

このプロジェクトは、歌枠（配信）のアーカイブを美しく、機能的に管理するための個人用データベースシステムです。

## ✨ Features
- 🦋 **Visual Interface**: 漆黒の背景に映える紫のネオンカラー。
- 🔍 **Smart Search**: カテゴリや日付による直感的な歌唱データのフィルタリング。
- ✒️ **Integrated Edit**: アプリ上から直接データベースを更新可能。
- 📜 **Change Logs**: 編集履歴を自動で記録し、データの整合性を保持。

## 🚀 How to use (Local)
1. リポジトリをクローンまたはダウンロード
2. `pip install -r requirements.txt`
3. `streamlit run app.py`
